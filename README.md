### 4. 机器学习入门 (`04_ML_Intro`)
#### 核心项目：手动实现一元线性回归（最小二乘法）
- **数学原理**：基于最小化均方误差（MSE）推导闭式解
  \[
  w = (X^T X)^{-1} X^T y
  \]
- **完整流程**：
  1. 数据生成：构造带噪声的线性数据集
  2. 模型训练：手动计算斜率 \(w\) 与截距 \(b\)
  3. 预测与评估：计算 MSE、\(R^2\) 指标
  4. 可视化：绘制原始数据散点 + 拟合直线（待环境完善后补充）
- **亮点**：不依赖 scikit-learn 等高级库，从数学公式出发理解模型本质
